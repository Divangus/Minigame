# Navesitas Navelin

## Description

This is a 1vs1 spaceship minigame

## Key Features

 - Local multiplayer (only 2)
 - Control with the same keyboard
 
## Controls

 - Player 1:
   - Move up: W
   - Move down: S
   - Shot: F
 - Player 2:
   - Move up: UP
   - Move down: DOWN
   - Shot: L
